# portfolio - Pendientes

## Decisiones

- Single-page vanilla portfolio. HTML + CSS + JS, sin build step. Hosted on GitHub Pages.
- Sidebar 240px + content 560px, collapsa a single column a 900px.
- Design system con CSS variables en `:root` de `styles.css`. Nunca hardcodear colores.
- Contenido viene de `../blog-articles/`. Las guidelines editoriales viven allá.
- En mantenimiento.

## Pendientes

Sin pendientes activos. Cuando se publique un artículo nuevo en blog-articles, sincronizarlo acá (HTML + media) y pushear.
